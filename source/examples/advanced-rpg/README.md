# Advanced RPG Example

A more complex Edwin RPG demonstrating:

- Multiple interconnected locations
- NPCs with branching dialogue and item inventories
- Conditional actions (gated behind flags / inventory)
- Custom game flags
- Multiple items with use callbacks

## Running

From the `source/` directory:

```bash
npm install
npm run dev
```

Then open the URL shown in the terminal and navigate to `examples/advanced-rpg/index.html`.
